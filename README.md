# Resume

Static bilingual resume site with Spanish and English entry pages, profile assets, and a minimal file structure that can be hosted anywhere.

## Stack
- Static web assets

## Local Development
```bash
python3 -m http.server 8080
```

## Key Commands
- `python3 -m http.server 8080` serve the static resume locally.

## Repository Layout
- `index.html` Spanish landing page for the resume.
- `cv-en.html` English version of the resume.
- `profile.png` profile image and static assets used by the pages.

## Deployment Notes
Because the repo is only static files, it can be deployed to GitHub Pages, Cloudflare Pages, Netlify, or any basic web server without a build step.
